# bbsDemo 前端接口文档

## 项目概述

bbsDemo 是一个基于 Vue 3 + TypeScript + Element Plus 的论坛前端项目，实现了用户认证、帖子管理、评论、点赞、收藏等核心功能。

## 技术栈

- **框架**: Vue 3 + TypeScript
- **UI 组件库**: Element Plus (按需引入)
- **路由**: Vue Router 4
- **HTTP 客户端**: Axios
- **构建工具**: Vite
- **状态管理**: Pinia

## 项目结构

```
src/
├── components/        # 公共组件
│   └── UserProfile.vue  # 用户个人资料组件（支持头像裁剪）
├── router/           # 路由配置
├── services/         # API 服务
│   ├── api.ts       # Axios 实例配置
│   └── userApi.ts   # API 接口封装
├── stores/          # Pinia 状态管理
├── utils/           # 工具函数
│   └── auth.ts      # Token 管理
├── views/           # 页面组件
│   ├── HomeView.vue      # 首页
│   ├── LoginView.vue     # 登录/注册
│   ├── ForumView.vue     # 论坛列表
│   └── PostDetailView.vue # 帖子详情
└── App.vue          # 根组件
```

## API 服务

### 基础配置

API 服务基于 Axios 封装，配置文件位于 `src/services/api.ts`：

- **基础 URL**: `http://localhost:8080`
- **超时时间**: 10000ms
- **Content-Type**: `application/json`

### 请求拦截器

自动在请求头中添加 Token：

```typescript
config.headers.Authorization = `Bearer ${token}`
```

### 响应拦截器

- 自动保存后端返回的 token
- 处理 401 未授权错误，自动跳转到登录页

## 接口列表

### 1. 认证相关 API (`authApi`)

| 方法 | 接口 | 描述 | 参数 |
|------|------|------|------|
| `POST` | `/api/auth/send-code` | 发送验证码 | `{ email: string, type: 'register' \| 'reset' }` |
| `POST` | `/api/auth/register` | 用户注册 | `{ email: string, password: string, code: string }` |
| `POST` | `/api/auth/login` | 用户登录 | `{ email: string, password: string }` |
| `POST` | `/api/auth/refresh` | 刷新令牌 | `{ refresh_token: string }` |
| `POST` | `/api/logout` | 用户登出 | `{ refresh_token: string }` |
| `POST` | `/api/logout-all` | 登出所有设备 | - |

**使用示例**:

```typescript
import { authApi } from '@/services/userApi'

// 登录
const response = await authApi.login({
  email: 'user@example.com',
  password: 'password123'
})

// 注册
await authApi.register({
  email: 'user@example.com',
  password: 'password123',
  code: '123456'
})

// 发送验证码
await authApi.sendCode({
  email: 'user@example.com',
  type: 'register'
})
```

| 方法 | 接口 | 描述 | 参数 |

| `PUT` | `/api/profile/nickname` | 更新昵称 | `{ nickname: string }` |

### 3. 帖子相关 API (`postApi`)

| 方法 | 接口 | 描述 | 参数 |
|------|------|------|------|
| `GET` | `/api/posts` | 获取帖子列表 | `{ page: number, page_size: number }` |
| `GET` | `/api/posts/:id` | 获取帖子详情 | `postId: string` |
| `POST` | `/api/posts` | 创建帖子 | `{ title: string, content: string }` |
| `PUT` | `/api/posts/:id` | 更新帖子 | `postId: string, { title: string, content: string }` |
| `DELETE` | `/api/posts/:id` | 删除帖子 | `postId: string` |
| `GET` | `/api/my/posts` | 获取我的帖子 | `{ page: number, page_size: number }` |

**使用示例**:

```typescript
import { postApi } from '@/services/userApi'

// 获取帖子列表
const posts = await postApi.getPosts({
  page: 1,
  page_size: 10
})

// 获取帖子详情
const post = await postApi.getPostDetail('post-id')

// 创建帖子
await postApi.createPost({
  title: '帖子标题',
  content: '帖子内容'
})
```

## Token 管理

Token 管理工具位于 `src/utils/auth.ts`：

```typescript
import { tokenManager } from '@/utils/auth'

// 设置 Token
tokenManager.setToken('your-token')

// 获取 Token
const token = tokenManager.getToken()

// 删除 Token
tokenManager.removeToken()

// 检查是否已登录
const isAuth = tokenManager.isAuthenticated()
```

## 错误处理

所有 API 请求都会返回 Promise，建议使用 try-catch 进行错误处理：

```typescript
try {
  const response = await postApi.getPosts({ page: 1, page_size: 10 })
  // 处理成功响应
} catch (error: any) {
  // 处理错误
  const errorMessage = error.response?.data?.error || '请求失败'
  console.error(errorMessage)
}
```

## 响应格式

标准响应格式：

```typescript
{
  "code": 200,        // 状态码
  "message": "success", // 消息
  "data": { ... },    // 数据
  "total": 100        // 总数（分页时）
}
```

## 分页参数

所有列表接口都支持分页：

- `page`: 当前页码（从 1 开始）
- `page_size`: 每页数量

返回数据包含 `total` 字段表示总数。

## 注意事项

1. 所有需要认证的接口会自动携带 Token
2. Token 过期会自动跳转到登录页
3. 文件上传需要设置 `Content-Type: multipart/form-data`
4. 建议使用 async/await 处理异步请求

## 头像裁剪功能

### 功能说明

用户个人中心支持头像裁剪功能，使用 `vue-cropper` 组件实现：

- ✅ 支持用户自由框选头像区域
- ✅ 1:1 比例裁剪
- ✅ 可拖动和调整裁剪框
- ✅ 实时预览裁剪效果
- ✅ 自动将裁剪后的图片上传

### 使用方式

1. 点击 "更换头像" 按钮
2. 选择图片文件
3. 在弹出的裁剪对话框中调整裁剪区域
4. 点击 "确认" 按钮完成上传
5. 或点击 "取消" 按钮放弃上传

### 技术实现

使用 `vue-cropper@next` 库，配置如下：

```typescript
const cropperOptions = {
  viewMode: 1,
  aspectRatio: 1,        // 1:1 比例
  autoCropArea: 0.8,     // 默认裁剪区域占图片的 80%
  dragMode: 'move',      // 拖动模式
  cropBoxMovable: true,  // 裁剪框可移动
  cropBoxResizable: true // 裁剪框可调整大小
}
```

裁剪后的图片会转换为 Blob 对象，然后通过 `uploadAvatar` 接口上传。
